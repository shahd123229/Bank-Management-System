Bank Management System (C++)
An advanced banking application designed to simulate real-world financial operations with a focus on data security and persistence.

Key Features:
    User Permission System: Role-based access control (Admin vs. User).
    Client Management: Full CRUD (Create, Read, Update, Delete) operations.
    Transaction Module: Handles deposits, withdrawals, and total balances.
    Data Persistence: Integrates with text files to store and retrieve data reliably.

Technical Highlights:
    File Handling: Expert use of fstream for database simulation.
    Security: Implemented bitwise/logic-based permissions.
    Clean Code: Structured using Functions and Structs for scalability.
